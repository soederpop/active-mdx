# Relationship Class

When you call `modelInstance.hasMany("Children")` or `modelInstance.belongsTo("Parent")` you will get an instance
of a Relationship class.

```javascript
class HasManyRelationship extends Relationship {}
class BelongsToRelationship extends Relationship {}

modelInstance.hasMany("Children").constructor.name // "HasManyRelationship"
modelInstance.belongsTo("Parent").constructor.name // "BelongsToRelationship"
```
